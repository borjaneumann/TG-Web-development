First multiplication and division and then addition and substraction

The increment operator (++) increments (adds one to) its operand and returns a value.

The decrement operator (--) decrements (subtracts one from) its operand and returns a value.

++ will show 4. -- will show 2.

The answer is 5. ++ are after the variable, so the returned value is shown without addition. the variable gets updated (4)

The answer is 6. ++ are after the variable, so the returned value is shown taking into account the updated valuable of the variable. The variable gets updated (4)
