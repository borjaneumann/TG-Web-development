  3. First-Line

    4. Het :: first-line CSS pseudo-element past stijlen toe op de eerste regel van een block-level element. Merk op dat de lengte van de eerste regel van veel factoren afhangt, waaronder de breedte van het element, de breedte van het document en de lettergrootte van de tekst.

    Voorbeeld

    Het eerste lijn moet rood zijn. Lorem ipsum dolor sit, amet consectetur adipisicing elit.   4. Grammar error

    This is an experimental technology

  5. Marker

    6. Het :: marker pseudo-element in CSS selecteert het markeringsvak van een element uit de lijst, dat meestal een opsommingsteken of een cijfer bevat. Het werkt op elk element of pseudo-element dat is geconfigureerd voor display: list-item, zoals het <li>:en

    element.

    Voorbeeld

    1. Duraznos
    2. Manzanas
    3. Ciruelas

  8. Selection

    9. Het pseudo-element selector :: selection CSS past regels toe op een gedeelte van een document dat door de gebruiker is gemarkeerd (bijvoorbeeld: selectie met de muis of een andere aanwijzer op een apparaat). Slechts een kleine subset van CSS-eigenschappen kan worden gebruikt in een :: selectieregel op de selector voor: kleur, achtergrond, achtergrondkleur en tekstschaduw. Merk op dat in het bijzonder achtergrondafbeelding wordt genegeerd, net als elke andere eigenschap.

    Voorbeeld

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Nobis quos nihil dignissimos modi omnis error eum nulla sunt excepturi animi.

  9. Slotted()

    10. The ::slotted() Het :: slotted () CSS-pseudo-element vertegenwoordigt elk element dat in een sleuf in een HTML-sjabloon is geplaatst (zie Sjablonen en sleuven gebruiken voor meer informatie). Dit werkt alleen bij gebruik binnen CSS geplaatst in een schaduw-DOM. Merk ook op dat deze selector geen tekstknooppunt selecteert dat in een sleuf is geplaatst; het richt zich alleen op feitelijke elementen.

    Geen Voorbeeld
