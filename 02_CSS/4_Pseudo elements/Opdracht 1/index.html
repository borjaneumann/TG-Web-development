Pseudo elementen

  1. After

    2. :: after maakt een pseudo-element aan dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap inhoud. Het is standaard inline. De content moet gegeven worden

  2. Backdrop

    3. Het :: backdrop CSS pseudo-element is een box ter grootte van de viewport die direct wordt weergegeven onder elk element dat in volledig scherm wordt gepresenteerd. Dit omvat zowel elementen die in volledige schermmodus zijn geplaatst met behulp van de Fullscreen API en <dialog> elementen.



  3. Before

    4. :: after maakt een pseudo-element aan dat het eerste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap inhoud. Het is standaard inline. De content moet gegeven worden

  4. Cue

    5. The ::cue CSS pseudo-element matches WebVTT cues within a selected element. This can be used to style captions and other cues in media with VTT tracks.
