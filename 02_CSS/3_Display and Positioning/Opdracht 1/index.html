Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html

Belangrijk! Elementen worden vervolgens gepositioneerd met behulp van de eigenschappen top, bottom, left en right. Deze eigenschappen werken echter niet tenzij de eigenschap position eerst is ingesteld. Ze werken ook anders, afhankelijk van de positiewaarde.

De eigenschap position specificeert het type positioneringsmethode dat voor een element wordt gebruikt. Er zijn vijf verschillende positiewaarden:

Antwoord: Er zijn 5 position properties.

  • Static

    • HTML-elementen worden standaard statisch gepositioneerd. Statisch geplaatste elementen worden niet be√Ønvloed door de eigenschappen boven, onder, links en rechts. Een element met positie: Static; is niet op een speciale manier gepositioneerd; het wordt altijd gepositioneerd volgens het normale verloop van de pagina:

  • Relative

    • Een element met positie: relative; is gepositioneerd ten opzichte van zijn normale positie. Door de eigenschappen boven, rechts, onder en links van een relative gepositioneerd element in te stellen, wordt het weg van zijn normale positie aangepast. Andere inhoud wordt niet aangepast om in een opening te passen die door het element wordt achtergelaten.

  • Fixed

    • Een element met positie: fixed; is gepositioneerd ten opzichte van de viewport, wat betekent dat het altijd op dezelfde plaats blijft, zelfs als de pagina wordt gescrolld. De eigenschappen boven, rechts, onder en links worden gebruikt om het element te positioneren. Een fixed element laat geen gat in de pagina waar het normaal gesproken zou zijn gelokaliseerd.

  • Absolute

    • Een element met positie: absolute; is gepositioneerd ten opzichte van de dichtstbijzijnde gepositioneerde voorouder (in plaats van gepositioneerd ten opzichte van de viewport, zoals vast). Echter; als een absolute gepositioneerd element geen gepositioneerde voorouders heeft, gebruikt het de hoofdtekst van het document en beweegt het mee met het scrollen van de pagina.

  • Sticky

    • Een element met positie: Sticky; wordt gepositioneerd op basis van de scrollpositie van de gebruiker. Een Sticky element wisselt tussen relative en absolute, afhankelijk van de scrollpositie. Het wordt relatief gepositioneerd totdat een bepaalde offset-positie in de viewport wordt bereikt - dan blijft het op zijn plaats "plakken" (zoals positie: vast).

Antwoord: De default positie is Static
