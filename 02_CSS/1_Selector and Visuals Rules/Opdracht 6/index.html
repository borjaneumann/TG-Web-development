Opdracht 6

  • Child selectors(>)

      • The child selector selecteerd alle elementen die children zijn van een element.

      Goed om binnen een element de kinderen snel te selecteren

      div > p

      Link naar child selector

      { background-color: yellow; }
  • Adjacent selectors(+)

      • The adjacent sibling selector selecteerd alle elementen die adjacent siblings zijn van het element.

      Goed om snel een van de siblings achter een element te selecteren

      Link naar adjacent selector

      div + p { background-color: yellow; }
  • General selectors(~)

      • The general sibling selector selecteerd alle elementen die siblings zijn van een element.

      Link naar general selector

      Goed om snel de siblings achter een element te selecteren

      div ~ p { background-color: yellow; }
